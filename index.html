    PART A — Learning (บทเรียนหลัก)

    หมวดนี้คือ “ห้องเรียน” — แต่ละบทใช้โครงเดียวกัน: Tools • How • When • Why • Misuse • Observation • Bridge • Summary

    CH01 — Market Structure (BOS / CHoCH)

    Tools Used

    • Structure Lines
    • Session HL
    • Fib 0–100

    How to Apply

    1. ทำโครงสร้าง swing high/low ล่าสุด
    2. รอ BOS (ยืนยันทิศ) และใช้ CHoCH เป็นสัญญาณเตือน
    3. หลีกเลี่ยงการ “เดา BOS” ก่อนเกิดจริง

    When to Use

    ก่อนเข้าสู่ Killzone เพื่อวาง Bias หลัก

    Why It Works

    เพราะ BOS คือหลักฐานของแรงที่ชนะจริง (order flow shift)

    Common Misuse

    • เข้าเร็วเพราะอยากเดา
    • CHoCH ในโซนผิด bias

    Observation

    หลัง BOS มักเห็น value zone ชัด: OB/FVG

    THEORY → PRACTICE BRIDGE

    BOS = ยืนยันพลัง • ปฏิบัติ = รอให้เกิดก่อน

    Summary

    • รอ BOS
    • ใช้ CHoCH เป็นสัญญาณระวัง
    • ห้ามเดาทิศ

    Chart Notes: ตัวอย่าง BOS/CHoCH + แนว FVG50%

    CH02 — Liquidity Map (PDH/PDL • Asia HL • EQH/EQL)

    Tools Used

    • PDH/PDL
    • Asia Range
    • EQH/EQL Marker

    How to Apply

    1. ตี PDH/PDL และกรอบ Asia
    2. รอ sweep → หา BOS → มองหาค่า FVG/OB

    When to Use

    ก่อนเข้า Killzone และช่วง confirm entry

    Why It Works

    ราคาไปหาสภาพคล่องเสมอ → sweep ก่อนกลับ

    Common Misuse

    • ไล่ช่วงกวาด (impulse)

    Observation

    หลัง sweep มักเกิด BOS สวน

    Bridge

    “ไล่หลังภาวะ” ไม่ใช่ “ไล่กวาด”

    Summary

    • ทำ Liquidity Map รายวัน
    • อย่าเข้าในช่วง sweep

    Chart Notes: Sweep High/Low + Asia Box + FVG 50%

    CH04 — Premium / Discount (Dealing Range)

    Tools

    • Fib 0–100
    • Dealing Range HL

    How

    1. ลาก Fib จาก swing low → high (ขาขึ้น) หรือ high → low (ขาลง)
    2. กำหนดฝั่งที่เหมาะสมต่อทิศ (value = discount ในขาขึ้น)

    When

    หลังยืนยัน Bias จากโครงสร้าง

    Why

    ซื้อถูก ขายแพง = Expectancy ดีขึ้น

    Misuse

    • Buy ใน Premium / Sell ใน Discount

    Bridge (ภาษา “จับมือทำ”)

    เราไม่เดา — ให้โครงสร้างบอกก่อนว่าแรงหลักไปทางไหน พอรู้ทิศ เราค่อยเข้า “เฉพาะฝั่ง value” เช่น discount ของ swing

    Summary

    • ตั้ง dealing range
    • เข้าเฉพาะ value
    • เชื่อมกับ OB/FVG

    Chart Notes: Premium/Discount + FVG50%

    PART B — Trade Setup Bible (จับมือทำทีละขั้น)

    ทุก Setup มีโครง **Identify → Stalk → Confirm → Execute → Manage → Reflect** และ **Entry/TP/SL Rules** ตาม SNF Score

    SETUP 1 — Sweep → BOS → FVG (Reversal)

    Step 0 — Preconditions

    • Bias จาก TF สูง (4H/1H) ชี้ชัด
    • มี Liquidity ใกล้เคียง (PDH/PDL/EQH/EQL)

    Step 1 — Identify

    รอ “กวาด” (sweep) โซนสภาพคล่อง → สังเกต displacement

    Step 2 — Stalk

    เฝ้าดูการเกิด BOS สวนทิศเดิม + หา FVG 50% ในฝั่งใหม่

    Step 3 — Confirm

    • Micro BOS ใน 5m/1m
    • OB สด ซ้อนใน FVG
    • เวลา: Killzone (London/NY)

    Step 4 — Execute

    • Entry: Tap เข้า FVG50% หรือ OB สด
    • SL: หลัง swing ล่าสุด (หรือหลัง OB)
    • TP: TP1 ที่ FVG เก่า / TP2 ที่โซน liquidity ถัดไป

    Step 5 — Manage

    • ย้าย BE ที่ +1R
    • Partial 30–50% ตามโครงสร้าง

    Step 6 — Reflect

    บันทึก: หลอก–ยืนยัน–เติมกำลัง–ปล่อยว่าง (ภาพรวม 4 สเตจ)

    Case: กวาด→BOS→เข้า FVG50% พร้อม Micro BOS

    SETUP 2 — OB-in-FVG (Continuation)

    Identify

    หา FVG ในเทรนด์หลัก แล้วมองหา OB สดที่ซ้อนใน FVG

    Confirm

    Tap + micro BOS

    Execute

    เข้า OB สดใน FVG, SL หลัง OB, TP ตาม value zone ถัดไป

    Manage

    ย้าย BE หลัง +1R

    PART C — Multi-Timeframe Execution Map (4H → 1H → 15m → 5m → 1m)

    ไล่จากภาพใหญ่สู่จุดเข้า: 4H วาง Bias • 1H ยืนยันค่าและเวลา • 15m หาโซน • 5m วาง RR • 1m กดปุ่ม

    TF 4H — Bias + โครงสร้าง + Liquidity Map

    TF 1H — Confirm Bias + Timing

    TF 15m — Setup Zone + Sweep Detection

    TF 5m — Execution Entry + RR Plan

    TF 1m — Candle Behavior / Precision Manage

    PART D — Risk • Journal • SNF Scoring • Compounding

    SNF Score (0–10 ต่อเกณฑ์)

    • S — Structure: BOS/CHoCH/Trend
    • N — Narrative: Liquidity สอดคล้อง
    • F — FVG/OB Value: เข้าใน value จริง
    • Timing — Session/ข่าว

    เข้าเฉพาะเคสที่คะแนนรวม ≥ 28/40

    Compounding Plan (12 สัปดาห์)

    เริ่มขนาดไม้ 0.25–0.5% ต่อ R • เป้าหมาย RR 1:3–1:5 • ปรับเพิ่มเมื่อชนะ 3 เคสติด
